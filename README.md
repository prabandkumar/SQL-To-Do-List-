# SQL To-Do List Project

## Description
A beginner-friendly To-Do List project using pure SQL.
This project helps understand CRUD operations in SQL.

## Technologies Used
- MySQL
- SQL

## Features
- Add tasks
- View tasks
- Update task status
- Delete tasks

## How to Run
1. Open MySQL
2. Copy and run `Todo list.sql`
3. Use `SELECT * FROM tasks;` to view tasks

## Learning Outcome
- SQL CRUD operations
- Database design basics
- WHERE, UPDATE, DELETE usage
